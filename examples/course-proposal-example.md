### 課程提案：Python 資料科學入門

#### 一、課程概述
- **課程名稱**：Python 資料科學入門
- **授課對象**：具備基礎程式設計概念的大學生或職場人士
- **課程類型**：線上授課（混合同步與非同步學習）
- **課程時長**：12 週，每週 3 小時（共 36 小時）
- **課程日期**：2026 年 3 月 - 5 月

#### 二、課程目標
學員完成本課程後將能夠：
1. 使用 Python 進行資料處理和分析
2. 運用 Pandas 和 NumPy 處理各種格式的資料集
3. 使用 Matplotlib 和 Seaborn 創建資訊豐富的視覺化圖表
4. 應用基礎統計分析方法解釋資料
5. 完成一個完整的資料分析專案

#### 三、課程大綱

**第 1-2 週：Python 基礎複習與環境建置**
- 主要內容：Python 語法複習、Jupyter Notebook、套件管理
- 時間分配：6 小時
- 學習重點：建立資料科學開發環境

**第 3-5 週：NumPy 與資料結構**
- 主要內容：NumPy 陣列操作、向量化運算、基礎線性代數
- 時間分配：9 小時
- 學習重點：高效能數值運算

**第 6-8 週：Pandas 資料處理**
- 主要內容：DataFrame 操作、資料清理、合併與重塑
- 時間分配：9 小時
- 學習重點：實務資料處理技巧

**第 9-10 週：資料視覺化**
- 主要內容：Matplotlib、Seaborn、圖表設計原則
- 時間分配：6 小時
- 學習重點：有效的資料呈現

**第 11-12 週：統計分析與期末專案**
- 主要內容：描述統計、假設檢定、專案實作
- 時間分配：6 小時
- 學習重點：綜合應用與專案開發

#### 四、教學方法
- **教學策略**：翻轉教室、問題導向學習
- **互動活動**：
  - 每週線上討論會
  - 即時程式碼示範
  - 同儕程式碼審查
- **實作練習**：
  - 每週課後作業
  - 真實資料集分析
  - 期末資料分析專案

#### 五、評估方式
- **形成性評估**（50%）
  - 每週作業：30%
  - 課堂參與與討論：10%
  - 同儕評量：10%
- **總結性評估**（50%）
  - 期末專案：40%
  - 專案報告：10%
- **評分標準**：
  - 程式碼品質與效能
  - 資料分析的深度與洞察
  - 視覺化的清晰度與美觀性
  - 文檔完整性

#### 六、所需資源
- **教材清單**：
  - 自編講義與 Jupyter Notebooks
  - Python for Data Analysis (Wes McKinney)
  - 精選線上資源與教學影片
- **設備需求**：
  - 個人電腦（Windows/Mac/Linux）
  - Python 3.8+ 環境
  - 線上學習平台帳號
- **參考資料**：
  - Kaggle 資料集
  - UCI Machine Learning Repository
  - 官方文檔（NumPy、Pandas、Matplotlib）

#### 七、預期成果
- **學習成效指標**：
  - 90% 學員能獨立完成資料分析流程
  - 80% 學員能產出專業品質的視覺化圖表
  - 學員專案作品可作為求職作品集
- **課程特色**：
  - 實戰導向，使用真實資料集
  - 業界講師客座分享
  - 建立個人專案作品集
- **後續發展建議**：
  - 進階課程：機器學習入門
  - 專題研究：特定領域資料分析
  - 實習機會：與合作企業媒合
