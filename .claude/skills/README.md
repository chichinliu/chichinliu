# Claude Code Skills

這個目錄包含自訂的 Claude Code Skills，用於協助完成特定任務。

## 可用的 Skills

### 課程提案生成器 (course-proposal)

協助你設計和撰寫完整的課程提案文檔。

**使用方式：**
```
@skill course-proposal
```

或在對話中輸入：
```
請幫我使用課程提案技能
```

**功能特色：**
- 互動式引導收集課程資訊
- 自動生成結構化提案文檔
- 提供專業的課程設計建議
- 符合 SMART 原則的目標設定
- 完整的課程大綱和教學規劃

**適用場景：**
- 設計新課程
- 撰寫課程提案給學校或機構
- 規劃線上或線下課程
- 培訓課程開發
- 工作坊和研討會規劃

## 如何使用 Skills

1. 在 Claude Code 中使用 `@skill` 指令
2. 輸入 skill 名稱（不含 .md 副檔名）
3. 跟隨 AI 的互動式引導完成任務

## 自訂 Skills

你可以在這個目錄中新增自己的 skill 文件：
- 使用 Markdown 格式 (.md)
- 清楚描述任務流程和輸出格式
- 提供具體的互動指引
