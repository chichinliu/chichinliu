# 第 7 章：客戶服務 Agent — 你的公關部門

> 「客戶不在乎是 AI 還是人在回覆，他們在乎的是有沒有被好好對待。」

---

## 7.1 客服對話 Agent：FAQ 與即時回覆

### 核心觀念
- 用 AI 處理 80% 的常規客服問題，讓你專注在 20% 需要人情味的場合

### 內容要點
- **設定客服 Agent 的四大要素**：
  1. **知識庫**：產品資訊、常見問題、退換貨政策、價格表
  2. **語氣設定**：親切友善 / 專業正式 / 活潑年輕（依品牌調性）
  3. **回覆邊界**：哪些問題可以直接回答、哪些需要轉交人類
  4. **升級機制**：什麼情況下要「轉接真人」

- **常見客服場景模板**：
  1. 產品詢問：「這個產品的功能是什麼？」
  2. 訂單查詢：「我的訂單到哪裡了？」
  3. 退換貨：「我想要退貨，怎麼處理？」
  4. 技術支援：「功能 X 不能用，怎麼辦？」
  5. 投訴處理：「我對服務很不滿意...」

- **投訴處理的特別設計**：
  - AI 先安撫情緒、確認問題
  - 超過處理範圍時優雅地轉交人類
  - 絕對不讓 AI 做承諾（退款、賠償等決策由人類做）

### 工具推薦
- ChatGPT Custom GPTs / Claude Projects
- Intercom / Zendesk AI / Tidio

## 7.2 社群管理 Agent：社群貼文與互動

### 核心觀念
- 社群經營需要持續輸出內容，AI 是你的內容引擎

### 內容要點
- **社群內容日曆 Agent**：
  - 規劃每週 / 每月的貼文主題
  - 自動配合節日、活動、趨勢
  - 輸出：內容日曆表格 + 每則貼文的初稿

- **各平台貼文 Agent**：
  - Facebook：長文 + 互動型貼文
  - Instagram：視覺優先 + Hashtag 策略
  - LinkedIn：專業知識分享 + 個人品牌
  - X (Twitter)：精簡有力 + 話題參與

- **社群互動 Agent**：
  - 留言回覆模板
  - 私訊回覆指引
  - 負面評論處理原則

- **注意事項**：
  - AI 生成的社群內容一定要人類最終審核
  - 避免過度自動化導致「機器人感」
  - 保持品牌的「人味」

## 7.3 Email 管理 Agent：郵件分類與自動回覆

### 核心觀念
- Email 是最耗時間的溝通工具之一，AI 能幫你大幅提效

### 內容要點
- **Email 分類 Agent**：
  - 自動將郵件分為：緊急 / 重要 / 一般 / 可忽略
  - 辨識郵件類型：客戶詢問 / 合作邀約 / 行銷信 / 內部通知

- **Email 回覆 Agent**：
  - 為不同類型的郵件準備回覆模板
  - AI 生成初稿，人類微調後發送
  - 常見場景：
    1. 會議邀請回覆
    2. 報價回覆
    3. 合作邀約回覆
    4. 客戶感謝信
    5. 延遲交付的道歉信

- **Email 摘要 Agent**：
  - 長郵件串的重點摘要
  - 待辦事項自動提取
  - 每日郵件摘要報告

### 工具推薦
- Gmail AI features / Outlook Copilot
- Superhuman / SaneBox
- 自建：ChatGPT + Zapier

## 7.4 實戰演練：建立一套自動化客戶服務系統

### 完整案例
- **情境**：你經營一間線上課程平台，需要處理學員的各種問題
- **系統設計**：

```
客戶訊息進入
    ↓
分類 Agent → 識別問題類型
    ↓
    ├─ 常見問題 → FAQ Agent 自動回覆
    ├─ 技術問題 → 技術客服 Agent 引導排除
    ├─ 退款要求 → 收集資訊 + 轉交人類決策
    └─ 投訴抱怨 → 安撫 Agent 回覆 + 轉交人類
    ↓
人類審核 → 處理需要人工介入的案例
    ↓
追蹤 Agent → 確認問題已解決、發送滿意度調查
```

- **完整呈現**：系統架構圖、每個 Agent 的設定、模擬對話範例
