# 第 4 章：策略規劃 Agent — 你的智囊團

> 「好的決策不是靠直覺，而是靠充分的資訊與系統化的分析。」

## 4.1 市場分析 Agent：趨勢洞察與競爭分析

### 核心觀念
- 市場分析不再是大公司的專利，AI 讓每個人都有專屬的市場研究員

### 內容要點
- **設定你的市場分析 Agent**：
  - 角色：資深市場研究分析師
  - 能力：產業趨勢分析、競爭對手追蹤、消費者洞察
  - 輸出格式：結構化報告（含數據、圖表建議、行動建議）

- **實用提示詞範本**：
  1. SWOT 分析模板
  2. 競爭對手分析模板
  3. 市場趨勢掃描模板
  4. 目標客群輪廓分析模板

- **使用注意事項**：
  - AI 的資料有時效性限制，需搭配即時搜尋
  - 數據要交叉驗證，不可全盤接受
  - AI 擅長框架分析，但洞察仍需人類判斷

### 工具推薦
- ChatGPT + Web Browsing / Perplexity / Claude
- 搭配 Google Trends、SimilarWeb 做數據驗證

## 4.2 企劃 Agent：專案規劃與方案設計

### 核心觀念
- 企劃 Agent 幫你從「想法」變成「可執行的計畫」

### 內容要點
- **設定你的企劃 Agent**：
  - 角色：專案規劃顧問
  - 能力：目標設定、里程碑規劃、資源配置、時程安排
  - 輸出格式：專案計畫書（含甘特圖建議、責任矩陣）

- **企劃 Agent 的三步驟工作法**：
  1. 目標釐清：透過提問幫你定義清楚的目標（SMART 原則）
  2. 結構設計：拆解為可執行的工作項目
  3. 計畫輸出：產生完整的專案計畫文件

- **實用提示詞範本**：
  1. 專案啟動計畫模板
  2. 活動企劃模板
  3. 產品規劃模板

## 4.3 決策輔助 Agent：利弊分析與風險評估

### 核心觀念
- 重要決策不應該靠「感覺」，讓 AI 幫你做系統化的分析

### 內容要點
- **設定你的決策輔助 Agent**：
  - 角色：企業策略顧問
  - 能力：利弊分析、風險評估、方案比較、情境推演
  - 輸出格式：決策分析報告（含評分矩陣、風險等級）

- **四種決策分析框架**：
  1. 利弊分析矩陣（Pros & Cons Matrix）
  2. 決策矩陣評分法（Weighted Decision Matrix）
  3. 風險評估表（Risk Assessment）
  4. 情境分析（Scenario Analysis）

- **實用提示詞範本**：
  1. 「我正在考慮 A 和 B 兩個選項，請幫我做系統化的利弊分析...」
  2. 「請幫我評估以下方案的風險...」

## 4.4 實戰演練：用策略 Agent 規劃一個新專案

### 完整案例
- **情境**：你是一位自由工作者，想推出一門線上課程
- **步驟**：
  1. 用市場分析 Agent 做線上課程市場調查
  2. 用企劃 Agent 設計課程規劃與上架計畫
  3. 用決策輔助 Agent 比較不同平台（Hahow / Udemy / 自架）
- **完整呈現**：每一步的提示詞、AI 回應、人類審核修改過程
