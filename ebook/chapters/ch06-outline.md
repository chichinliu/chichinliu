# 第 6 章：資料分析 Agent — 你的數據部門

> 「數據不會說話，但 AI 可以幫你翻譯數據背後的故事。」

---

## 6.1 資料蒐集 Agent：網路爬蟲與資訊彙整

### 核心觀念
- 資料蒐集是分析的基礎，AI 能大幅加速這個過程
- 不需要寫程式也能做資料蒐集

### 內容要點
- **三種資料蒐集方式**：
  1. **AI 搜尋彙整**：用 Perplexity / ChatGPT 搜尋並彙整特定主題資訊
  2. **結構化資料擷取**：從網頁、PDF、報告中提取關鍵數據
  3. **問卷與訪談分析**：用 AI 分析開放式問卷回覆或訪談逐字稿

- **設定資料蒐集 Agent**：
  - 角色：研究助理
  - 指令要點：明確指定蒐集範圍、資料格式、來源限制
  - 輸出格式：表格化、含來源引用、標註可信度

- **品質把關**：
  - 要求 AI 標註資料來源
  - 交叉驗證關鍵數據
  - 注意資料時效性

### 工具推薦
- Perplexity AI（搜尋 + 彙整）
- ChatGPT with Browsing（網頁資料擷取）
- Claude（長文件分析）

## 6.2 分析 Agent：數據處理與視覺化

### 核心觀念
- AI 讓「不會 Excel 公式」的人也能做數據分析

### 內容要點
- **設定資料分析 Agent**：
  - 角色：數據分析師
  - 能力：數據清理、趨勢分析、異常偵測、統計摘要

- **四種常見分析任務**：
  1. **描述性分析**：「這組數據告訴我們什麼？」
     - 基本統計量、分布、趨勢
  2. **比較分析**：「A 和 B 有什麼差異？」
     - 前後對比、群組比較
  3. **相關性分析**：「什麼因素影響了結果？」
     - 變數之間的關係
  4. **預測分析**：「接下來可能會怎樣？」
     - 基於趨勢的推估

- **不需要寫程式的分析方法**：
  - 上傳 CSV / Excel 到 ChatGPT（Code Interpreter）
  - 用自然語言描述你想看的分析
  - AI 自動產生圖表和洞察

- **視覺化指引**：
  - 哪種數據用哪種圖表（長條圖/折線圖/圓餅圖/散點圖）
  - 如何請 AI 產生美觀的圖表

## 6.3 報告 Agent：自動生成分析報告

### 核心觀念
- 從原始數據到可交付的報告，AI 可以自動化 80% 的工作

### 內容要點
- **設定報告 Agent**：
  - 角色：商業分析報告撰寫師
  - 輸入：分析結果、目標讀者、報告用途
  - 輸出：結構化報告（含摘要、圖表、建議）

- **報告模板設計**：
  1. 執行摘要（重點結論，一頁以內）
  2. 數據概覽（關鍵指標 Dashboard）
  3. 詳細分析（圖表 + 說明）
  4. 洞察與建議（可行動的建議）
  5. 附錄（數據來源、方法說明）

- **從分析到報告的 Agent 串聯**：
  - 分析 Agent 產出數據洞察 → 報告 Agent 撰寫報告 → 編輯 Agent 潤飾

## 6.4 實戰演練：用數據 Agent 團隊完成市場調研

### 完整案例
- **情境**：你經營一間小型電商，想了解「台灣保健食品市場」的現況與機會
- **Agent Team 分工**：

```
蒐集 Agent → 市場規模、競品、消費趨勢
    ↓
分析 Agent → 數據整理、趨勢分析、機會識別
    ↓
報告 Agent → 生成完整市場調研報告
    ↓
人類審核   → 確認洞察、補充在地知識、做出決策
```

- **完整呈現**：提示詞 → AI 輸出 → 人類回饋 → 最終報告
