# 第 5 章：內容創作 Agent — 你的創意部門

> 「AI 不會取代創作者，但善用 AI 的創作者會取代不用 AI 的創作者。」

---

## 5.1 寫作 Agent：文章、文案、報告撰寫

### 核心觀念
- 寫作 Agent 不是「代筆」，是你的「初稿產生器」和「靈感激發器」

### 內容要點
- **設定你的寫作 Agent**：
  - 角色設定要素：語氣、風格、目標讀者、品牌聲調
  - 上下文提供：品牌指引、過去的範文、競品參考

- **七種常見寫作場景與模板**：
  1. 部落格文章 / SEO 文章
  2. 社群貼文（Facebook / Instagram / LinkedIn）
  3. 電子報 / Email 行銷
  4. 產品文案 / 銷售頁
  5. 商業報告 / 提案書
  6. 新聞稿 / 公關稿
  7. 教學內容 / 課程講義

- **提升寫作品質的技巧**：
  - 提供風格範例（Few-shot Learning）
  - 分段撰寫，不要一次要求太長的內容
  - 先要大綱，確認後再展開
  - 給予明確的字數、段落、格式要求

---

## 5.2 編輯 Agent：校對、潤飾、風格統一

### 核心觀念
- 寫作和編輯是兩種不同的能力，應該用不同的 Agent 處理

### 內容要點
- **設定你的編輯 Agent**：
  - 角色：資深文字編輯
  - 任務類型：
    1. 校對（錯字、語法、標點）
    2. 潤飾（語句流暢度、節奏感）
    3. 風格統一（確保全文語氣一致）
    4. 精簡（刪除冗詞、提升精準度）

- **編輯 Agent 的分層使用**：
  - Level 1：基礎校對（錯字、標點）
  - Level 2：語句潤飾（流暢度、可讀性）
  - Level 3：風格審核（品牌一致性、語氣適合度）

- **實用提示詞範本**：
  1. 「請校對以下文章，只修改錯字和語法問題，不要改變原意...」
  2. 「請潤飾以下文章，讓它更流暢、更有節奏感，保持原有風格...」
  3. 「以下是我們的品牌風格指引，請檢查這篇文章是否符合...」

## 5.3 多媒體 Agent：圖片生成、簡報製作

### 核心觀念
- 文字內容 + 視覺內容 = 完整的創作交付

### 內容要點
- **圖片生成 Agent**：
  - 工具：Midjourney / DALL-E / Stable Diffusion / Canva AI
  - 常見場景：文章配圖、社群圖片、產品展示圖、概念圖
  - 提示詞技巧：風格描述、構圖指示、色調控制
  - 注意事項：版權問題、品牌一致性

- **簡報製作 Agent**：
  - 工具：Gamma / Beautiful.ai / ChatGPT + PowerPoint
  - 流程：用 AI 生成大綱 → 填充內容 → 設計視覺
  - 提示詞：指定頁數、每頁重點、視覺風格

- **影片腳本 Agent**（進階）：
  - 短影音腳本撰寫
  - YouTube 影片大綱
  - Podcast 節目規劃

## 5.4 實戰演練：用創作 Agent 團隊完成一篇行銷企劃

### 完整案例
- **情境**：為一間咖啡店撰寫周年慶的完整行銷企劃
- **Agent Team 分工**：

```
步驟 1：策略 Agent → 行銷策略與目標
步驟 2：寫作 Agent → 活動文案初稿
         ├─ 社群貼文文案（5 篇）
         ├─ EDM 內容
         └─ 店內海報文案
步驟 3：編輯 Agent → 校對與風格統一
步驟 4：視覺 Agent → 配圖指引與簡報
步驟 5：人類審核 → 最終確認
```

- **完整呈現**：每步的提示詞、AI 輸出範例、人類修改意見
