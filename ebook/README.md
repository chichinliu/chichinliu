# AI 智能工作術 - 電子書專案

## 專案概述

本電子書《AI 智能工作術：打造你的 Agent Team》旨在教導讀者如何運用 AI Agent 團隊協作，
建構自己的智能工作室，全面提升工作效率與創造力。
