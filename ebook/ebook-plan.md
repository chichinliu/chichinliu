## 二、核心理念

> **AI 智能工作術** = 一個人 + 一組 AI Agent Team = 一間公司的產能

本書的核心觀點：每個人都可以透過組建自己的 AI Agent 團隊，打造一間「一人公司」
級別的智能工作室。不需要寫程式，不需要技術背景，只需要學會如何指揮你的 AI 團隊。

## 四、每章結構模板

每一章遵循統一的結構：

```
章節標題
├── 章節引言（情境故事或案例引入）
├── 核心概念說明（2-3 個重點觀念）
├── 步驟教學（圖文並茂的操作指引）
```

## 五、寫作風格指引

| 面向 | 要求 |
|------|------|
| **語氣** | 專業但親切，像一位經驗豐富的朋友在分享 |
| **用詞** | 避免過度技術術語，必要時附上簡明解釋 |
| **案例** | 每章至少 2 個貼近真實工作場景的案例 |
| **圖片** | 每章搭配操作截圖、流程圖、架構圖 |
| **互動** | 每章設計 1-2 個讀者可立即操作的練習 |
| **長度** | 每章約 4,000-6,000 字 |

## 六、目標與差異化

### 本書獨特價值
1. **團隊思維**：不只教單一工具，而是教如何組建和管理 AI 團隊
2. **實戰導向**：每章都有可操作的實戰演練
3. **系統化**：提供完整的工作室建構框架，不只是零散技巧
4. **接地氣**：面向中文使用者，案例與情境貼近本地職場

### 預期成果
- 讀者讀完後能建立自己的 AI Agent Team
- 至少能自動化 50% 的重複性工作
- 具備持續擴展和優化 AI 工作室的能力
