# 各章節寫作指引

本文件提供每一章的具體寫作指引，供 Agent Team 在撰寫時參考。

---

## 通用寫作規範

### 格式規範
- **標題層級**：章名用 H1、節名用 H2、小節用 H3，最多到 H4
- **段落長度**：每段 3-5 句，避免大段落
- **列表**：操作步驟用有序列表、特點列舉用無序列表
- **程式碼/提示詞**：用 code block 包裹，附上語言標記
- **圖片**：用 `![描述](路徑)` 格式，圖片描述要清楚

### 術語對照表（持續更新）

| 英文 | 繁體中文 | 備註 |

## 各章寫作指引

### 第 1 章：歡迎來到 AI 智能工作室

**目標**：點燃讀者對 AI 智能工作室的想像與興趣

**開場建議**：
- 用一個生動的對比場景開場：傳統工作方式 vs. AI 智能工作室
- 例如：一位自由工作者如何用 AI Agent Team 在一天內完成過去一週的工作量

**關鍵內容**：
- 清楚定義什麼是「AI 智能工作室」
- 強調這不是取代人類，而是「增強」人類能力
- 展示本書的完整學習路線圖（配合視覺化圖表）

**避免事項**：
- 不要過度技術化，第一章要讓所有人都能讀懂
- 不要列舉太多工具名稱，保持焦點在概念上

**參考案例**：一人創業者 / 小型工作室 / 遠端工作者

---

### 第 2 章：認識你的 AI Agent 團隊

**目標**：讓讀者理解 AI Agent 與傳統 AI 工具的本質差異

**關鍵內容**：
- Agent vs. Tool 的清楚對比（ChatGPT 當工具用 vs. 當 Agent 用）
- Agent 的四大核心能力用生活化比喻解釋
- 「團隊」的概念：為什麼多個專職 Agent 比一個萬能 Agent 更好

**實作元素**：
- 讀者練習：把自己目前的工作拆解成可以分配給不同 Agent 的任務

---

### 第 3 章：AI 智能工作室的運作架構

**目標**：建立讀者對「系統化」AI 工作流的認知框架

**關鍵內容**：
- 三層架構圖（策略層 / 執行層 / 品管層）——需搭配架構圖
- 人類的不可取代角色：創意決策、價值判斷、情感連結
- 「需求 → 拆解 → 分配 → 執行 → 審核 → 交付」的完整流程

**實作元素**：
- 讀者練習：選一個自己的工作專案，畫出 AI 協作流程圖

---

### 第 4 章：策略規劃 Agent — 你的智囊團

**目標**：教讀者如何用 AI 做策略思考與規劃

**關鍵內容**：
- 市場分析的提示詞範本（SWOT、競爭分析、趨勢分析）
- 企劃撰寫的 Agent 設定與使用方法
- 決策輔助：如何讓 AI 幫你做利弊分析

**實戰演練**：
- 完整案例：用 AI Agent 為一個新產品做市場調研與企劃書
- 包含具體提示詞、AI 回應範例、人類審核修改的過程

---

### 第 5 章：內容創作 Agent — 你的創意部門

**目標**：教讀者建立 AI 內容創作的系統化流程

**關鍵內容**：
- 寫作 Agent 的角色設定技巧（語氣、風格、目標讀者）
- 編輯 Agent 的校對與潤飾流程
- 多媒體 Agent 的使用（圖片生成、簡報製作）
- 多 Agent 串聯：寫作 → 編輯 → 視覺化的完整流程

**實戰演練**：
- 完整案例：用 Agent Team 完成一篇社群行銷企劃
  - 寫作 Agent 撰寫初稿
  - 編輯 Agent 校對潤飾
  - 視覺 Agent 產生配圖

### 第 6 章：資料分析 Agent — 你的數據部門

**目標**：教讀者如何用 AI 處理資料分析任務

**關鍵內容**：
- 用 AI 做資料蒐集與整理（不需寫程式碼的方法）
- 請 AI 分析數據並產生洞察
- 自動化報告生成

**實戰演練**：
- 完整案例：用 Agent 團隊完成一份市場調研報告
- 注意：要展示「不會寫程式也能做數據分析」的方法

**關鍵內容**：

**實戰演練**：
- 完整案例：為一個小型電商建立客服 Agent 系統
- 包含：FAQ 知識庫建立 → 對話腳本設計 → 測試與優化

## 品質檢查清單

每章完成後，依照以下清單逐項檢查：

- [ ] 章節標題清楚且吸引人
- [ ] 開場引言能引起讀者興趣
- [ ] 核心概念解釋清楚，無歧義
- [ ] 至少包含 2 個貼近真實場景的案例
- [ ] 實戰演練步驟具體、可操作
- [ ] 提示詞範例可直接複製使用
- [ ] 文字無錯字、語法正確
- [ ] 術語使用統一，符合對照表
- [ ] 章節間邏輯銜接順暢
- [ ] 字數在 4,000-6,000 字之間
- [ ] 圖表說明完整
- [ ] 常見問題 Q&A 有實用價值
- [ ] 本章重點回顧精準到位
